**工具安装**  
  
1、系统要求  
  
安装Sirius Scan前请确保您的系统满足以下要求：  
```
Docker Engine：版本 20.10.0 或更高版本
Docker Compose：V2
Git：最新稳定版本
内存：最低 4GB
存储空间：10GB可用磁盘空间
```  
  
  
2、安装步骤  
  
1. 克隆存储库  
```
git clone https://github.com/SiriusScan/Sirius.git
cd Sirius
```  
  
2. 启动服务  
```
docker compose up -d
```  
  
3.   
访问Web界面  
```
浏览器中打开：http://localhost:3000
使用默认凭据登录：用户名:admin，密码：password
```
